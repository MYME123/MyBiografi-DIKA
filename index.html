Web Server:
Web server adalah perangkat keras atau perangkat lunak yang bertugas menyimpan, mengelola, dan mengirimkan halaman web kepada pengguna. Ketika seseorang mengakses suatu website, permintaan dikirimkan ke web server yang kemudian merespons dengan mengirimkan halaman web yang diminta. Contoh web server populer meliputi Apache, Nginx, dan Microsoft IIS.

Domain:
Domain adalah alamat unik yang digunakan untuk mengidentifikasi suatu situs web di internet. Misalnya, "www.namadomain.com". Domain membantu pengguna untuk mengingat alamat situs web tanpa perlu menghafal alamat IP numerik yang sesungguhnya dari server. Domain terdiri dari dua bagian utama: top-level domain (TLD), seperti .com, .org, .net, dan nama domain itu sendiri.

Hosting:
Hosting adalah layanan penyimpanan dan pemeliharaan data (termasuk file website) di server yang terhubung ke internet. Saat Anda membuat sebuah website, Anda perlu menyimpan file-file tersebut di server hosting agar dapat diakses oleh pengguna secara online. Hosting dapat dibagi menjadi beberapa jenis, seperti shared hosting (berbagi sumber daya dengan situs web lain), virtual private server (VPS), dan dedicated hosting (server yang ditugaskan khusus untuk satu situs).

Hubungan antara Web Server, Domain, dan Hosting:
1. Ketika seseorang mengetikkan alamat domain suatu website (misalnya, www.contoh.com) di browser, permintaan itu dikirim ke server DNS (Domain Name System).
2. Server DNS kemudian mengarahkan permintaan tersebut ke IP address yang sesuai dengan domain tersebut.
3. Setelah mendapatkan IP address, browser mengirim permintaan ke web server yang diidentifikasi oleh IP tersebut.
Web server merespons dengan mengirimkan file-file website ke browser pengguna, dan browser kemudian menampilkan halaman web tersebut.
